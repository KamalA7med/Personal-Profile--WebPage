# Personal Profile Web Page

## Project Overview
This is a simple personal profile web page that presents basic information in a clean and organized layout.  
It includes a header with name and photo, an about section, a skills section, a media element, and a contact form.  
The page is designed to be responsive, user-friendly, and visually appealing.

## Features
- **Header:** Displays your name and profile photo.
- **About Section:** A short description about yourself.
- **Skills Section:** Lists your skills with proficiency levels.
- **Media Element:** Includes a video or image showcasing your work.
- **Contact Form:** Allows visitors to contact you easily.
- **Responsive Design:** Works well on smaller screens and mobile devices.
